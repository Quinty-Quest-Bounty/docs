## Repositories

| Repo | Stack | Purpose |
|------|-------|---------|
| [sc-quinty](https://github.com/Quinty-Quest-Bounty/sc-quinty) | Solidity, Hardhat, OpenZeppelin | Smart contracts, ABIs, deploy scripts |
| [fe-quinty](https://github.com/Quinty-Quest-Bounty/fe-quinty) | Next.js 15, React 19, Wagmi, Privy | Frontend web application |
| [be-quinty](https://github.com/Quinty-Quest-Bounty/be-quinty) | NestJS, Supabase, JWT | Authentication API and user profiles |
| [indexer-quinty](https://github.com/Quinty-Quest-Bounty/indexer-quinty) | Ponder, PostgreSQL | Blockchain event indexer with GraphQL |
| [landing-quinty](https://github.com/Quinty-Quest-Bounty/landing-quinty) | Vite, React, GSAP | Marketing landing page |
| [docs](https://github.com/Quinty-Quest-Bounty/docs) | Mintlify | This documentation site |

## Data Flow

```
User action in frontend
  --> Wagmi sends transaction to smart contract on Base Sepolia
  --> Contract emits events
  --> Indexer picks up events and stores in PostgreSQL
  --> Frontend queries indexer's GraphQL API for display

User login in frontend
  --> Privy authenticates user
  --> Frontend calls backend /auth/sync-profile
  --> Backend upserts profile in Supabase
  --> Backend returns JWT cookie
  --> Frontend uses cookie for authenticated API calls
```
