## ABIs

Contract ABIs are exported from the `sc-quinty` repository:

```
sc-quinty/exported-abis/
  Quinty.json              # Bounty contract ABI
  Quest.json               # Quest contract ABI
  all-abis.json            # All ABIs in one file
  constants.ts             # Addresses, enums, USDC address
```

The `constants.ts` file exports TypeScript types and addresses:

```typescript
import { BASE_SEPOLIA_ADDRESSES } from "./exported-abis/constants";

// BASE_SEPOLIA_ADDRESSES.Quinty
// BASE_SEPOLIA_ADDRESSES.Quest
// BASE_SEPOLIA_ADDRESSES.QuintyReputation
// BASE_SEPOLIA_ADDRESSES.QuintyNFT
```
